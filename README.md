# BIES-Toy-Compiler

## 🇪🇸 Español

Durante el desarrollo del proyecto en Node.js, se utilizó ChatGPT como herramienta de apoyo para la consulta e integración de utilidades clave como Jest, Lint y Prettier. También se empleó para obtener soporte en tareas relacionadas con el uso de la consola, gramática del lenguaje y el diseño de funciones auxiliares que complementan las funcionalidades principales del evaluador.

Asimismo, se aprovechó como recurso de aprendizaje para comprender y aplicar algoritmos fundamentales como el Shunting Yard de Dijkstra, dentro del flujo del proyecto.

---

## ⚙️ Proceso de Build Automatizado Completo

1. Clonar el repositorio y navegar a la raíz del proyecto (bies):
```bash
cd bies
```

2. Ejecutar el proceso de build automatizado (compila BIESC y BIESVM):
```bash
npm run buildGlobal
```

---

## ▶️ Ejecución individual

### 🟦 Archivos .bies

```bash
cd biesc
biesc file.bies
```

### 🟩 Archivos .basm

```bash
cd biesvm
biesvm file.basm
```

---

## 🧾 Nota importante

- BIESC_OUT → Muestra detalles de la ejecución del bytecode en BIESC  
- BIESC_ERROR → Muestra errores durante la ejecución en BIESC  
- BIESVM_OUT → Muestra el bytecode compilado en BIESVM  
- BIESVM_ERROR → Muestra errores durante la ejecución en BIESVM  

---

## 🇺🇸 English

During the development of the Node.js project, ChatGPT was used as a support tool for consulting and integrating key utilities such as Jest, Lint, and Prettier. It was also used to obtain assistance with console usage, language grammar, and the design of helper functions that complement the main evaluator functionality.

Additionally, it served as a learning resource to understand and apply fundamental algorithms such as Dijkstra’s Shunting Yard algorithm within the project workflow.

## ⚙️ Complete Automated Build Process

1. Clone the repository and navigate to the project root (bies):
```bash
cd bies
```

2. Run the automated build process (compiles both BIESC and BIESVM):
```bash
npm run buildGlobal
```

## ▶️ Individual Execution

### 🟦 .bies files

```bash

biesc file.bies
```

### 🟩 .basm files


---

## 🧾 Important Notes

- BIESC_OUT → Provides details about bytecode execution in BIESC  
- BIESC_ERROR → Provides error details if any occur in BIESC  
- BIESVM_OUT → Shows compiled bytecode in BIESVM  
- BIESVM_ERROR → Provides error details if any occur in BIESVM
